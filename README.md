# Quiz application

A Simple quiz game application where the user/player answer a bunch of multiple-choice question 

## How to run
to run install either the simple folder(jar version) or the source folder(source code version)
execute the .bat file and keep everything in one folder 

## Requirement
-java 8 or higher

## Features
-differente ssubject
-recent quiz done 
-history of quiz
-keep track of your stats
-login/register and a guest mode availble
